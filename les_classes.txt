Créer sa classe
On creer une classe pour représenter une entité ou une idée ou un concept.

Structure 

public                  class                  nomdeclasse 
(droit d'accés)         mot clef class         généralement le nom du fichier (1 classe par fichiers)

Les attributs
    Variables qui appartiennent à une classe.

Les méthodes
    Fonctions qui appartiennent à une classe.

this fait reference a l'instance courante
super fais reference a la classe "mere"
final : 
        sur classe -> signifie qu'une classe ne peut pas être étendue
        sur methode -> signifie qu'une méthode ne peut pas être réécrit
        sur attribut -> signifie qu'un attribut ne peut pas être modifié
extends permet d'étendre une classe
